Migre para Java 7 Agora!

Há 15 anos atrás, nascia uma incrível linguagem de programação. Ela se chamava Java. Agora estamos em 2013, o mundo evoluiu bastante e Java evoluiu com ele. Para as versões 4, 5 and 6 de Java, é hora de dizer adeus!

Migre para Java 7 Agora!

Este site é dedicado a ajudar você a migrar para Java 7. Dessa forma, nós vamos ver o uso de Java 4, 5 & 6 cair para menos de 1% em todo o mundo. Com isso, milhões de usuários estarão mais seguros e milhões desenvolvedores se tornarão mais produtivos."

Ainda não Está Pronto para Migrar?

Se, por alguma razão especial, você ainda não está pronto para migrar, Oracle está pronta para oferecer o suporte que você precisa para manter o seu negócio e seus clientes seguros.

Sobre este Site

Esta é uma iniciativa da comunidade de Java Champions; dos Grupos de Usuários Java, representados pelos seus JUG Leaders; e do programa Adopt OpenJDK. Este é o principal local para obter informações sobre a situação da migração do suporte Java a navegadores web. Nosso objetivo é encorajar usuários finais a adotar a última e melhor versão da tecnologia Java.
